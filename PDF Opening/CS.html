Computer Science Notes

1. Introduction to Computer Science

Computer Science is the study of algorithms, data structures, programming, and system design. It includes areas such as artificial intelligence, cybersecurity, and software engineering.

2. Programming Languages

Programming languages are used to write software and applications.

Low-Level Languages: Assembly, Machine Language.

High-Level Languages: Python, Java, C++, JavaScript.

Object-Oriented Programming (OOP): Focuses on classes and objects (e.g., Java, C++).

3. Data Structures

Data structures organize and store data efficiently.

Array: A fixed-size collection of elements.

Linked List: A sequence of nodes connected by pointers.

Stack: Last In, First Out (LIFO) structure.

Queue: First In, First Out (FIFO) structure.

Graph: A collection of nodes connected by edges.

4. Algorithms

Algorithms are step-by-step procedures for solving problems.

Sorting Algorithms: Bubble Sort, Merge Sort, Quick Sort.

Searching Algorithms: Binary Search, Linear Search.

Graph Algorithms: Dijkstras Algorithm, BFS, DFS.

5. Operating Systems

An operating system manages hardware and software resources.

Types of OS: Windows, macOS, Linux.

Processes and Threads: OS handles multitasking using process scheduling.

Memory Management: Virtual memory, paging, segmentation.

6. Databases

Databases store, retrieve, and manage data efficiently.

Relational Databases: Use tables (SQL, MySQL, PostgreSQL).

NoSQL Databases: Flexible schema (MongoDB, Firebase).

Normalization: Reduces redundancy in relational databases.

7. Computer Networks

Computer networking enables communication between devices.

Network Types: LAN, WAN, MAN.

Protocols: TCP/IP, HTTP, FTP, DNS.

OSI Model: 7 layers - Physical, Data Link, Network, Transport, Session, Presentation, Application.

8. Cybersecurity

Cybersecurity protects computers and networks from attacks.

Types of Threats: Malware, Phishing, Ransomware.

Encryption: Protects data using algorithms like AES, RSA.

9. Artificial Intelligence and Machine Learning

AI enables computers to learn from data and make decisions.

Types of AI: Weak AI (e.g., Siri), Strong AI (e.g., AGI).

Machine Learning: Algorithms that learn from data (Supervised, Unsupervised Learning).

10. Software Development

Software Development involves designing, coding, and maintaining applications.

Development Models: Waterfall, Agile, DevOps.

Version Control: Git, GitHub.
